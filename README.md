# AXUDI Website

A modern, accessible and responsive website developed for **AXUDI (Asociación Xuvenil de Diabetes)**, a youth association focused on supporting and connecting young people living with diabetes.

This project was designed and developed from the ground up with a strong focus on **user experience, accessibility, responsive design, maintainability and SEO best practices**. The goal was to create a welcoming digital space where visitors can easily discover the association, stay informed about activities and news, and connect with the community.

---

## Project Goals

* Create a professional and trustworthy online presence for the association.
* Deliver an excellent experience across mobile, tablet and desktop devices.
* Prioritize accessibility and inclusive design.
* Build a scalable content system that allows non-technical members to manage news independently.
* Ensure long-term maintainability through reusable components and clean project architecture.

## Key Features

### User Experience & Interface Design

* Custom visual identity based on AXUDI's branding.
* Mobile-first design approach.
* Clear content hierarchy and intuitive navigation.
* Consistent design system with reusable UI components.
* Smooth interactions and subtle animations.
* Optimized layouts for different screen sizes and orientations.

### Accessibility

* Semantic HTML structure.
* Keyboard-friendly navigation.
* Accessible forms and interactive elements.
* Meaningful image alternative text.
* Visible focus states.
* Reduced-motion support for users who prefer fewer animations.
* Attention to color contrast and readability.

### SEO & Performance

* Route-specific metadata management.
* Canonical URLs.
* Open Graph and social sharing metadata.
* Optimized image delivery using modern formats.
* Clean semantic markup for improved indexing.
* Performance-focused frontend architecture.

### Content Management

* CMS-powered news management workflow.
* Non-technical editors can create and update content independently.
* Search, filtering and pagination for news articles.
* Fallback handling for missing images and content.

### Contact & Community

* Newsletter subscription integration.
* Contact form with submission feedback states.
* Association membership call-to-action.
* Easy access to activities, events and community information.

## Local Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Run lint checks:

```bash
npm run lint
```
